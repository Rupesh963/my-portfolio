# My Portfolio

A modern and responsive developer portfolio built with Next.js, React, TypeScript, and Tailwind CSS. It showcases my education, technical skills, projects, and contact information in a clean and interactive interface.

## Features
- Responsive design for desktop and mobile
- Smooth animations with Framer Motion
- Dedicated sections for About, Education, Projects, Skills, and Contact
- Easy to customize and extend

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
